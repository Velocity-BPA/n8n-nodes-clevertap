# Automate Your Marketing Workflows with the New CleverTap Node for n8n

We're excited to announce that Velocity BPA has released a new community node that brings CleverTap's powerful customer engagement platform directly into your n8n workflows!

## The Challenge

Marketing teams and developers working with CleverTap often find themselves manually triggering campaigns, updating user profiles, or pulling analytics data through custom scripts or API calls. This creates bottlenecks in workflows and limits the ability to create sophisticated, event-driven marketing automation.

## Introducing n8n-nodes-clevertap

Our new CleverTap node eliminates these friction points by providing seamless integration between CleverTap and your n8n automation workflows. Whether you're managing user profiles, tracking events, orchestrating multi-channel campaigns, or coordinating messaging across email, SMS, and push notifications, this node puts CleverTap's capabilities at your fingertips within the visual n8n interface.

## Key Features

**User Profile Management**: Create, update, and retrieve user profiles directly from your workflows. Sync customer data from your CRM, database, or any other connected system without writing a single line of code.

**Event Tracking**: Automatically send custom events to CleverTap based on triggers from other applications. Track user behaviors, conversions, and interactions in real-time.

**Campaign Orchestration**: Trigger and manage campaigns programmatically. Connect CleverTap campaigns to external events, schedules, or business logic.

**Multi-Channel Messaging**: Coordinate messaging across email, SMS, push notifications, and in-app messages through unified workflows.

## Getting Started

Installing the CleverTap node is simple. In your n8n instance, run:


npm install n8n-nodes-clevertap


After installation, restart n8n, and you'll find the CleverTap node available in your node palette. Connect your CleverTap account using your API credentials, and you're ready to start automating!
